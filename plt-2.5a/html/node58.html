Known bugs

Several minor bugs that were present in earlier versions of plt have been fixed. Those that have not yet been fixed are listed here; if you know of any others, or if you find any incompatibilities between the current version of plt and a classic version, please let me (george@mit.edu) know!

  • Text scaling in xplt does not match that in lwplt, with the result that there can be minor differences in the size and spacing of plot labels when rendered on-screen and on paper.

  • This bug list is probably incomplete!
